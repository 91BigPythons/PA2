1.create two instances in chameleon using our own key: rachelgogogo.pem
2. modify config file to access two VMs through bastion
3. install mininet in two VMs
4. set up vxlan:

####
ssh og_team1_vm1
ssh og_team1_vm2
sudo ufw disable
####

get IPs of VM1(192.168.5.220 ) and VM2(192.168.5.105) from chameleon
in VM1:
sudo ip link delete vxlan0
sudo ip link add vxlan0 type vxlan id 100 local 192.168.5.220 remote 192.168.5.105 dev ens3 dstport 4789
sudo ip addr add 233.233.233.1/24 dev vxlan0
sudo ip link set vxlan0 up
sudo mn -c
sudo ip route add 10.10.2.0/24 via 233.233.233.1 dev vxlan0

in VM2:
sudo ip link delete vxlan0
sudo ip link add vxlan0 type vxlan id 100 local 192.168.5.105 remote 192.168.5.220 dev ens3 dstport 4789
sudo ip addr add 233.233.233.2/24 dev vxlan0
sudo ip link set vxlan0 up
sudo mn -c
sudo ip route add 10.10.2.0/24 via 233.233.233.2 dev vxlan0

5. set up mininet:
in another terminal in VM1:
sudo mn --nat -i 10.10.1.0/24

in another terminal in VM2:
sudo mn --nat -i 10.10.2.0/24
